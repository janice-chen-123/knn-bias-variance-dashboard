# Monte Carlo Study of the Bias-Variance Tradeoff in k-NN

An interactive Shiny app for exploring the bias-variance tradeoff in k-NN through Monte Carlo simulation in 1D and 2D settings.

## Live Demo
[Launch the Shiny App](https://connect.posit.cloud/janice-chen-123/content/019ce7b4-9ea2-6114-47b2-89c584870684)

## Key Features
- Interactive controls for training sample size, noise level, number of neighbors, and grid size
- 1D and 2D simulation settings for studying fitted functions and error behavior
- Monte Carlo comparison of empirical and theoretical error curves
- 3D surface visualization for selected 2D outputs

## Tech Stack
R, Shiny, ggplot2, plotly, tidyverse

## Repository Structure
- `shiny-app/`: source code for the Shiny website
- `vignettes/`: project vignette and AI statement
- `tests/`: test files
